# Sample ZH Project

Ez egy nagyon egyszerű kis Node.js projekt, ami demonstrálja a tesztfuttatás módját amit majd a ZH-n is használnotok kell
hogy megnézzétek jól működik-e a megoldásotok.

## Előfeltételek
* Node.js és NPM fel van telepítve (utóbbi automatikusan feltelepül a Node.js-szel együtt).

## Útmutató
1. Nyissatok egy tetszőleges command line-t (Windows CMD, PowerShell, Git Bash, etc)
2. Navigáljatok el a `sample-zh-project` könyvtárba.
3. Adjátok ki az `npm install` parancsot, hogy a futtatáshoz szükséges NPM package-ek feltelepüljenek.
4. Adjátok ki az `npm test` parancsot hogy a tesztek lefussanak és kapjatok egy visszajelzést az eredményről és kódlefedettségről.

Értelemszerűen ezen lépések és parancsok nem csak egy parancssoron keresztül futtathatóak, a különböző kódszerkesztők integrálódnak a Node-dal és NPM-mel.
Egy parancssor viszont független attól hogy ki milyen IDE-t használ és jól reprodukálható minden hallgatónál.
Természetesen a ZH és a projekt beadandó fejlesztése közben úgy használjátok az eszközöket ahogy nektek legjobban kézre esik: a lényeg az eredmény.
* IntellIJ IDEA: https://www.jetbrains.com/help/idea/vitest.html#ws_vitest_running_tests
* Vitest dokumentáció IDE integrációkhoz: https://vitest.dev/guide/ide.html
