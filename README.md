## 👨🏻‍💻 ZH Tudnivalók
* Időpont: 2026. 03. 17. 16:00
* Téma: HTML, CSS & JavaScript.
* Módja: online.
* Időtartam: 45 perc - 1 óra.
* Szükséges szoftverek: Node.js, Git, kedvenc kódszerkesztőd (VS Code, WebStorm, IntelliJ IDEA, stb), tetszőleges böngésző
* Közvetlenül az óra előtt közzéteszünk Teams-en egy dedikált GitHub repository-t, ahol megtaláljátok majd a feladatokat.
* A kódbázist mindenki klónozza le a saját gépére és kövesse a benne lévő útmutatásokat.
* A megoldásokat pull request-ek formájában várjuk az eredeti repository felé nyitva.
* A repository tartalmaz majd előre megírt teszteket a JavaScript feladatokhoz, ezeknek sikeresen kell lefutniuk. A futtatás mikéntjéről a repoban találtok majd instrukciókat.
* A rendelkezésre álló idő után a pull request nyitás lehetősége megszűnik.
* A sikeres ZH a pull request-ek és a tesztfuttatások kiértékelésének függvénye.

## ❓ FAQ
### Lehet használni AI-alapú kódgeneráló eszközöket?
Igen, az egyetemi szabályzatnak megfelelő eszközöket lehet alkalmazni mind a ZH megoldása mind a projektmunka fejlesztése közben.
Arra kérünk mindenkit hogy a használt promptokat / skill-eket / workflow-kat / hook-okat / plan-eket, mindent ami input az AI-nak, azt valamilyen formában csatoljátok
a megoldásotokhoz (például .txt vagy .md fájlokként).

### Meg van szabva milyen kódszerkesztővel dolgozhatok?
Nincs, nyugodtan használjátok amit lehet és kézre esik nektek.

### Valami nem tiszta, kihez fordulhatok segítségért?
Teams-en nyugodtan keressetek bennünket oktatókat és segítünk.
